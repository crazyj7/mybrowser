# WPF Browser

간단한 웹 브라우저 프로그램입니다.

## 주요 기능
- 다중 탭 브라우징
- 북마크 관리 (드래그 앤 드롭으로 순서 변경 가능)
- 홈페이지 설정
- 탭/창 상태 저장 및 복원
- 단축키 지원

## 시스템 요구사항
- Windows OS
- .NET 8.0
- WebView2 Runtime

## 단축키
- Ctrl+W: 현재 탭 닫기
- Ctrl+B: 북마크 패널 토글
- F1: 프로그램 정보

## 빌드 방법
1. Visual Studio 2022 이상 버전 설치
2. 프로젝트를 클론 또는 다운로드
3. WpfBrowser.sln 파일을 열어서 빌드
